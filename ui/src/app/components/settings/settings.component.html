A time-based TRNG generates randomness by measuring execution time differences between identical loops, which are influenced by system-level unpredictability like CPU load and scheduling. The randomness is primarily classical, with quantum effects (e.g., thermal noise and oscillator stability) contributing less than 0.001%. This makes it a practical but not quantum-driven method for generating random numbers.
